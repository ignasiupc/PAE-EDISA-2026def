# Legacy

Esta carpeta contiene el sistema anterior que ya no es el flujo principal del
proyecto.

- `main.py` y `src/`: aplicacion antigua de localizacion en almacen.
- `config/`: configuracion del flujo antiguo.
- `scripts/`: utilidades asociadas a ese flujo.
- `assets/warehouse_markers/png/`: marcadores generados para el almacen.
- `prototypes/`: scripts `detector_simple.py` y `detector_pose.py`.

Si trabajas sobre la version actual del proyecto, usa `../detector_3d.py`.
