# Ejemplos y prototipos

Esta carpeta agrupa scripts auxiliares y pruebas que no forman parte del flujo
principal de `main.py`.

- `detector_simple.py`: detección básica y visualización mínima.
- `detector_pose.py`: estimación de pose con captura por proximidad.
- `detector_3d.py`: prototipo avanzado con Kalman y visualización 3D.
- `calibration/calibrar_camara.py`: utilidad de calibración para los prototipos.
- `config/markers_3d.json`: mapa de marcadores usado por `detector_3d.py`.
